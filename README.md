# Sayzy

**One line. One voice.**

Sayzy is a free, open-source AAC (Augmentative and Alternative Communication) app that turns a drawing, a letter, or a single tap into instant speech — built for people who can't speak, made to work in under a second.

## Why Sayzy

Most communication tools ask a nonverbal user to search through menus or symbol grids to say something simple. Sayzy flips that: draw a quick shape, tap a saved shortcut, or type a sentence — and it's spoken instantly. Every feature lives behind one uncluttered drawing screen, and nothing shows up until it's needed.

## Features

- **Instant draw-to-speech** — draw a symbol or letter, hear it spoken back immediately
- **Handwriting recognition** — understands rough, imperfect drawings, not just perfect shapes
- **Custom shortcuts** — lock any drawing to any phrase, permanently, until you change it
- **Multi-letter codes** — set shorthand like "KH" for a full saved phrase
- **Type-to-speech** — type anything, hear it instantly
- **Live captions** — see exactly what's being spoken, in real time
- **Speech history** — a timestamped log of everything said
- **Family voice recording** — hear saved phrases in a loved one's real voice
- **Shopping & task lists** — build a list at home, play it aloud item by item
- **Fully offline** — no internet required for core features
- **Built for every hand** — large touch targets and haptic feedback
- **Free & open source** — built in the open, forever
